# Machine-learning
Tarea Machine learning


La evaluación del módulo consiste en participar en un concurso de datos abierto de forma permanente en DrivenData, donde el objetivo es predecir el estado de bombas de agua a partir de unas 50 variables de distintos tipos (numéricas y categóricas) y con dificultades reales como valores faltantes, categorías de alta cardinalidad y clases desbalanceadas; la variable objetivo es multiclase (no binaria) y las bombas pueden estar en tres estados: funcional, no-funcional y necesita_reparación. Para participar debes registrarte en DrivenData y trabajar en la competición “Pump it Up: Data Mining the Water Table”; descargarás los datasets que proporciona el concurso: el conjunto de entrenamiento (train), las etiquetas (labels) que contienen el target y se deben unir al train, y el conjunto de test sobre el que realizarás las predicciones, además de un archivo de ejemplo de submission que indica el formato de entrega. El flujo de trabajo consiste en entrenar tus modelos con train + labels, generar predicciones para el test y crear un archivo de envío con dos columnas: el id de cada bomba del test y la clase predicha; esas predicciones se suben en la sección de submissions del concurso, teniendo en cuenta que solo se permiten 3 envíos diarios, por lo que conviene iterar con tiempo. Es obligatorio respetar las reglas del concurso, especialmente la prohibición de usar datos externos (APIs u otras fuentes fuera del material oficial del concurso). Como entregable para la evaluación, debes presentar un Notebook de Python claro y bien estructurado (texto y código) que documente las etapas seguidas con el modelo con el que obtuviste tu mejor resultado, incluyendo comentarios que expliquen cada paso, las transformaciones realizadas, el tratamiento de missing values, el manejo de variables categóricas, el entrenamiento y validación del modelo y la generación del archivo de submission, y puedes añadir también la puntuación (score) conseguida; el notebook debe leerse como un informe entendible y ordenado para alguien de negocio que revisa el trabajo.
